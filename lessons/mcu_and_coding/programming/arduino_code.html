From Blocks to Text: Introducing Arduino Code

You've learned to speak robot with blocks; now it's time to see the written language behind them! Arduino code is the text-based language that gives you even more control and flexibility in programming your robots. In this lesson, we'll bridge the gap between visual blocks and real code, uncovering how your block creations translate into the Arduino language and how you can use the Arduino IDE.

Introducing the Arduino IDE üíª

The Arduino IDE (Integrated Development Environment) is the software you'll use to write, compile (check for errors), and upload your Arduino code to a real, physical Arduino board. It's a free and open-source application that's available for Windows, macOS, and Linux. Think of it as your main coding workshop for your physical robot brains.

Key Parts of the Arduino IDE:

  • Code Editor Window: This is where you type your Arduino code. It includes features like syntax highlighting to make the code easier to read.
  • Toolbar: Contains buttons for common actions:
    • Verify (Check Mark): Compiles your code to check for errors without uploading it.
    • Upload (Right Arrow): Compiles and sends your code to the connected Arduino board.
    • New: Creates a new sketch (Arduino program).
    • Open: Opens an existing sketch.
    • Save: Saves your current sketch.
  • Serial Monitor: Opens a window to display serial data from your Arduino board‚Äîgreat for debugging.
  • Message Area: Displays feedback from the IDE like compilation errors, upload success messages, and serial output.

The Structure of an Arduino Program üß±üß±

Every Arduino program, also called a sketch, has a basic structure with two main, essential functions: 

  void setup() {
    // put your setup code here, to run once:
  }

  void loop() {
    // put your main code here, to run repeatedly:
  }
  • void setup(): This function runs only once when your Arduino board powers up or is reset. It's where you put code to initialize things, like setting up the communication speed, or telling the Arduino which pins will be inputs and which will be outputs.
  • void loop(): This function runs continuously, over and over again, after the setup() function has finished. This is where the main logic of your program goes ‚Äì the instructions that will be repeated to control your robot or electronic project.

Here's a simple example of the blinking LED code again, showing the setup() and loop() functions: 

    // This code will blink an LED connected to pin 13

    void setup() {
      // Set digital pin 13 as an OUTPUT
      pinMode(13, OUTPUT);
    }

    void loop() {
      // Turn the LED ON (send HIGH voltage)
      digitalWrite(13, HIGH);
      // Wait for 1000 milliseconds (1 second)
      delay(1000);
      // Turn the LED OFF (send LOW voltage)
      digitalWrite(13, LOW);
      // Wait for 1000 milliseconds (1 second)
      delay(1000);
    }
  • void setup(): This function runs only once when your Arduino board powers up or is reset. It's where you put code to initialize things, like setting up the communication speed, or telling the Arduino which pins will be inputs and which will be outputs.
  • void loop(): This function runs continuously, over and over again, after the setup() function has finished. This is where the main logic of your program goes ‚Äì the instructions that will be repeated to control your robot or electronic project.

In the setup() function, we tell the Arduino that pin 13 will be used to send signals out to the LED. In the loop() function, we tell it to turn the LED on, wait, turn it off, wait, and then repeat that forever.

  • pinMode(pin, mode);: This function is used in setup() to configure a specific pin as either an INPUT (to receive signals) or an OUTPUT (to send signals).
    • pinMode(7, INPUT); sets pin 7 as an input pin.
    • pinMode(13, OUTPUT); sets pin 13 as an output pin.
  • digitalWrite(pin, value);: This function sets a digital pin to either HIGH (5V) or LOW (0V). It's used to control output components.

  • delay(milliseconds);: This function pauses the program execution for the specified number of milliseconds.

Writing Simple Arduino Programs ‚úçÔ∏è

Let's walk through writing the blinking LED program directly in the Arduino IDE: 

    void setup() {
      // Set digital pin 13 as an output
      pinMode(13, OUTPUT);
    }
    
    void loop() {
      // Turn the LED on
      digitalWrite(13, HIGH);
      // Wait for 1 second (1000 milliseconds)
      delay(1000);
      // Turn the LED off
      digitalWrite(13, LOW);
      // Wait for 1 second
      delay(1000);
    }

Explanation of Each Line:

  • void setup() {: This marks the beginning of the setup() function.
  • pinMode(13, OUTPUT);: This line tells the Arduino that digital pin number 13 will be used to send signals out (to control the LED).
  • }: This curly brace marks the end of the setup() function.
  • void loop() {: This marks the beginning of the loop() function, which will run repeatedly.
  • digitalWrite(13, HIGH);: This line sends a HIGH voltage (5V) to digital pin 13, which will turn the LED connected to it ON.
  • delay(1000);: This line pauses the program for 1000 milliseconds, which is equal to one second. The LED will stay ON for this time.
  • digitalWrite(13, LOW);: This line sends a LOW voltage (0V) to digital pin 13, turning the LED OFF.
  • delay(1000);: This line pauses the program for another second, so the LED stays OFF for one second.
  • }: This curly brace marks the end of the loop() function. The program will then go back to the beginning of the loop() and repeat the process.

Compiling and Uploading (In Principle):

  1. Connect your Arduino Uno board to your computer using a USB cable.
  2. Open the Arduino IDE and paste or type the code into the editor window.
  3. Select the correct board: Go to Tools > Board and choose "Arduino Uno".
  4. Select the correct port: Go to Tools > Port and select the serial port that your Arduino is connected to (it will usually be labeled with "Arduino" or "USB Serial Device").
  5. Compile the code: Click the "Verify" button (the check mark). The IDE will check your code for errors. If there are any, they will be displayed in the message area.
  6. Upload the code: If your code compiles without errors, click the "Upload" button (the right arrow). The IDE will compile the code again and then send it to your Arduino board. You should see some messages in the message area indicating the upload process. Once it's done, the LED connected to pin 13 on your physical Arduino board should start blinking!
